# NLP Emotion Detection Web Application
A Flask based web application that detects emotions in text using IBM Watson NLP library.
## Features
- Detects five emotions:anger, disgust, fear, joy and sadness
- Identifies the dominant emotion in any given text
- REST API endpoint for emotion analysis
- Error handling for blank or invalid inputs
## Tech Stack
- Backend:Python, Flask
- AI/NLP:IBM Watson NLP Emotion Predict API
- Testing:Python unittest
- Code Quality:Pylint 10/10 score
